# AlphaWingEX-Remake
Just trying to recreate one of my favorite J2ME game in morern technology like OpenGL with C++

A 2.5D space shooter remake of classic J2ME mobile game Alpha Wing EX by Microspace,
rebuilt from the ground up in C++ with modern graphics and audio.

This is a personal project and midstone submission for Game Development at Humber College. 
The original game was a childhood favorite, so I am trying to bring to the new hardware and
audience to enjoy.

How to run this on your PC?
	- Download and install Visual Studion on windows (other OS!, you are on your own for now)
	- Setup GameDev library (find the GameDev.7z inside, extract it directly to your C drive "C:\GameDev")
	- Clone/Downoad, Open ComponentFramework.sln in VS and run, "F3" for the testing scene

How to play?
	- W A S D for movement
	- Spacebar/M1 to shoot
